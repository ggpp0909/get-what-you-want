## 개요

- 아무리 잘 구성된 사이트라고 하더라도, 이용자들이 없다면 무용지물이라고 생각합니다.
- 따라서 저희 팀은 **"사람들이 어떻게 하면 저희 사이트를 많이 이용할 수 있을지"**를 중점에 두었습니다.
  - 사용자의 입장에서 필요한 기능은 넣고, 불필요한 기능은 굳이 넣지 않는다.
